The `@tambo-ai/react/v1` package is the next generation of Tambo's React SDK, featuring a streaming-first architecture, explicit thread management, and React Query integration.

## Installation

```bash
npm install @tambo-ai/react
```

## Import Path

V1 APIs are imported from the `/v1` subpath:

```tsx
import {
  TamboV1Provider,
  useTamboV1,
  useTamboV1ThreadInput,
} from "@tambo-ai/react/v1";
```

## Overview

The V1 SDK is organized around a simplified provider hierarchy with React Query for data fetching:

```tsx
import { TamboV1Provider } from "@tambo-ai/react/v1";

function App() {
  return (
    <TamboV1Provider
      apiKey={process.env.NEXT_PUBLIC_TAMBO_API_KEY}
      userKey={userId}
      components={
        [
          /* your components */
        ]
      }
      tools={
        [
          /* your tools */
        ]
      }
    >
      <YourApp />
    </TamboV1Provider>
  );
}
```

Inside the provider, use hooks to access Tambo functionality:

```tsx
import { useTamboV1, useTamboV1ThreadInput } from "@tambo-ai/react/v1";

function Chat() {
  const { messages, isStreaming, startNewThread } = useTamboV1();
  const { value, setValue, submit } = useTamboV1ThreadInput();

  return (
    <div>
      {messages.map((msg) => (
        <Message key={msg.id} message={msg} />
      ))}
      <input
        value={value}
        onChange={(e) => setValue(e.target.value)}
        onKeyDown={(e) => e.key === "Enter" && submit()}
      />
    </div>
  );
}
```

## Key Differences from Current SDK

The V1 SDK introduces several changes from the current (`@tambo-ai/react`) SDK:

| Aspect           | Current SDK                | V1 SDK                                                                                    |
| ---------------- | -------------------------- | ----------------------------------------------------------------------------------------- |
| Import path      | `@tambo-ai/react`          | `@tambo-ai/react/v1`                                                                      |
| Main hook        | `useTambo()`               | `useTamboV1()`                                                                            |
| User scoping     | `contextKey` prop          | `userKey` prop                                                                            |
| Thread input     | `useTamboThreadInput()`    | `useTamboV1ThreadInput()`                                                                 |
| Message format   | Simple structure           | Content blocks (text, component, tool_use, etc.)                                          |
| Component state  | `useTamboComponentState()` | `useTamboV1ComponentState()` (bidirectional sync)                                         |
| Generation stage | `GenerationStage` enum     | `streamingState.status` (typed as [`RunStatus`](/reference/react-sdk-v1/types#runstatus)) |
| Data fetching    | Context-based              | React Query hooks                                                                         |

## What's Not Yet Supported

<Callout type="error" title="Not Yet Available">
  The following features from the current SDK are not yet available in V1:

Interactable components are not yet available in V1.

This includes the `withTamboInteractable` HOC and all related hooks.

</Callout>
