## Migrating from `useTamboStreamingProps`

<Callout type="warning" title="Deprecated">
  `useTamboStreamingProps` is deprecated and will be removed in 1.0.0.
</Callout>

**Before:**

```tsx
const [state, setState] = useTamboComponentState("myState", {
  title: "",
  body: "",
});
useTamboStreamingProps(state, setState, { title, body });
```

**After:**

```tsx
const { streamStatus } = useTamboStreamStatus<MyProps>();
const [state, setState] = useTamboComponentState(
  "myState",
  { title: "", body: "" },
  title && body ? { title, body } : undefined,
);
```

The new approach gives you streaming status for free and is easier to reason about.
