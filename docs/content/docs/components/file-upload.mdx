## Usage

### Basic File Upload

```
const mockUploader = async (files, onProgress) => {
  // Simulate upload progress
  let progress = 0;
  const interval = setInterval(() => {
    progress += 10;
    files.forEach((file) => {
      onProgress(file.name, progress);
    });
    
    if (progress >= 100) {
      clearInterval(interval);
    }
  }, 200);
  
  return files.map(file => `https://example.com/uploaded/${file.name}`);
};

function BasicExample() {
  return (
    <FileUpload 
      uploader={mockUploader}
      maxSizeMB={5}
    />
  );
}
```

### Multiple Images Only

```
function MultipleImagesExample() {
  return (
    <FileUpload 
      uploader={mockUploader}
      accept="image/*"
      multiple={true}
      maxSizeMB={2}
    />
  );
}
```

### Document Upload

```
function DocumentUploadExample() {
  return (
    <FileUpload 
      uploader={mockUploader}

      maxSizeMB={10}
    />
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `uploader` | `(files: File[], onProgress: (id: string, progress: number) => void) => Promise<string[]>` | **required** | Function to handle file uploads with progress callback |
| `accept` | `string` | `undefined` | Accepted file types (e.g., '.jpg,.png' or 'image/*') |
| `multiple` | `boolean` | `false` | Allow multiple file selection |
| `maxSizeMB` | `number` | `undefined` | Maximum file size in megabytes |
| `disabled` | `boolean` | `false` | Disable the component |
| `className` | `string` | `undefined` | Additional CSS classes |
| `children` | `ReactNode` | `undefined` | Custom content for drop zone |

## Features

- ✅ **Drag & Drop**: Intuitive file selection via drag and drop
- ✅ **File Validation**: Type and size validation before upload
- ✅ **Progress Tracking**: Real-time upload progress with visual indicators
- ✅ **Cancel/Retry**: Users can cancel ongoing uploads or retry failed ones
- ✅ **Multiple Files**: Support for single or multiple file selection
- ✅ **Accessibility**: Full keyboard navigation and screen reader support
- ✅ **Responsive**: Works seamlessly across all device sizes

## Accessibility

The FileUpload component includes comprehensive accessibility features:

- **ARIA Labels**: Proper labeling for screen readers
- **Keyboard Navigation**: Full keyboard support with focus management
- **Progress Announcements**: Upload progress is announced to assistive technologies
- **Error Communication**: Clear error states and messaging
- **Focus Management**: Logical tab order and focus indicators
