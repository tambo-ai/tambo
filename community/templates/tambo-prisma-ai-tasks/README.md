# Tambo + Prisma Starter

A minimal Next.js starter template demonstrating how to use **Tambo** with a **database-backed tool layer** powered by **Prisma**.

This template shows how an AI assistant can safely **create, read, update, and delete records** using Tambo tools instead of hallucinating data.

## What This Template Demonstrates

- Using **Tambo** with **Next.js App Router**
- Registering **database-backed tools** with Tambo
- Safe **AI-driven CRUD operations** using Prisma
- Preventing hallucinations by enforcing tool usage
- Clean separation between UI, AI logic, and database logic

This is a **starter template**, not a polished app.
