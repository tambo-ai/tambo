# Tambo Template (Next.js App Router)

A minimal Next.js App Router template with Tambo integration.

## Prerequisites

- Node.js 18+
- Tambo API key from [tambo.co/dashboard](https://tambo.co/dashboard)

## Setup

1. `npm install`
2. `npx tambo init` (or create `.env.local` with `NEXT_PUBLIC_TAMBO_API_KEY=your-key`)
3. `npm run dev`
4. Open [localhost:3000](http://localhost:3000)

## What this demonstrates

Basic Tambo provider setup and thread input usage in a Next.js App Router application.
