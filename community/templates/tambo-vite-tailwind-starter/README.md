# tambo-vite-tailwind-starter

A professional starter template for building **dynamic React UIs with Tambo**.

This template is designed for hackathons and includes:

- **Real-Time AI Preview Animation**: Hover over component cards to see animated â€œAI generatingâ€ loader dots.
- **Clickable Component Preview Cards**: Signup Form, Analytics Table, and Info Card with hover animations and selection.
- **Dark Mode Toggle**: Fully functional dark mode that changes backgrounds, text, and cards.
- **Prompt Input + History**: Save last 5 prompts and reuse with a click.
- **Loader Animation**: Shows while generating components or simulating AI responses.
