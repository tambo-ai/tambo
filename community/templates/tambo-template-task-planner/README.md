# Tambo Task Planner Starter Template

A beginner friendly starter template that demonstrates how to use Tambo to turn overwhelming goals into clear, actionable tasks.

## Who is this for
- Students
- Firsttime Tambo users
- Developers looking for a simple AI integration example

## What does this template do?
You enter a goal that feels overwhelming.
The app converts it into a step-by-step task list.

## Tech Stack
- React
- Vite
- Tambo (API-ready)

## Quick Start

```bash
git clone https://github.com/Serenehue/tambo.git
cd tambo/community/templates/tambo-template-task-planner
npm install
npm run dev
