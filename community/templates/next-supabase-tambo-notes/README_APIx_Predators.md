Next.js + Supabase + Tambo Notes Starter
A minimal starter template for building AI-powered apps where a Tambo assistant can read from and write to a real database using Supabase.

This template is intentionally simple so you can use it as a foundation for your own product.

---

What this template demonstrates
Next.js App Router for modern routing and layouts
Supabase Auth + Postgres for user authentication and persistent data
Tambo AI integration using official APIs and tool registration
An AI assistant that can:
Create notes for the authenticated user
List existing notes from the database
Secure, per-user data access using Row Level Security (RLS)
